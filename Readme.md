# Go-shell

A shell in Golang.

## Features

- [x] Run commands
- [x] Run commands with arguments
- [x] Run commands with pipes
- [ ] Run commands with redirections
- [ ] Run commands in background
- [ ] Support Shell Scripting
- [ ] Env variables
- [ ] Support for aliases

## Installation

### Build to binary

```bash
go build -o go-shell
```

### Run

```bash
./go-shell
```
